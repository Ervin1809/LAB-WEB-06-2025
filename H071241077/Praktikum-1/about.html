Bio

Halo! Nama saya Diesty Mendila Tappo, seorang mahasiswa aktif di Universitas Hasanuddin, program studi Sistem Informasi. Saya memiliki minat dalam mempelajari teknologi web dan desain, serta sangat antusias untuk mengembangkan keterampilan saya di bidang ini. Saya percaya bahwa kerja keras itu penting, tapi istirahat itu wajib. Di waktu luang, saya suka menjelajahi tren terbaru dalam desain web dan mencoba berbagai alat desain seperti Figma.


Saya telah menyelesaikan beberapa proyek yang melibatkan pembuatan situs web responsif menggunakan HTML dan CSS. Saya juga memiliki pengalaman dalam bekerja dengan tim untuk mengembangkan solusi kreatif yang memenuhi kebutuhan klien. Saya selalu berusaha untuk belajar hal baru dan meningkatkan kemampuan saya agar dapat memberikan hasil terbaik dalam setiap proyek yang saya kerjakan.


Terima kasih telah mengunjungi portofolio saya. Saya berharap dapat terhubung dengan Anda dan berbagi lebih banyak tentang perjalanan saya di dunia teknologi dan desain.
